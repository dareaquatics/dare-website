Motion & Animation

One custom easing curve does almost everything. Reveals use IntersectionObserver — elements start hidden and animate in when they enter the viewport.

Core Easing

--ease-out
cubic-bezier(0.16, 1, 0.3, 1)

0.2s ease
Card hover
0.25s ease
Nav background
0.35s ease
Drawer slide
0.42s cubic-bezier(0.16, 1, 0.3, 1)

Scroll Reveal Variants

/* Add these classes to any element; JS adds .is-revealed on intersection */
/* Threshold: 0.12 for most, 0.1 for stats, 0.6 for value panels */
/* --reveal-distance: 32px (default) */

.reveal--fade   → opacity 0 → 1 (no transform)
.reveal--up     → translateY(32px) → 0
.reveal--left   → translateX(-52px) → 0
.reveal--right  → translateX(52px) → 0
.reveal--scale  → scale(0.94) → 1
.reveal--blur   → blur(14px) + translateY(20px) → clear
.reveal--rotate → perspective(600px) rotateX(14deg) translateY(28px) → 0
.reveal--clip   → clip-path: inset(0 0 100% 0) → inset(0)
.reveal--wave   → skewY(4deg) + translateY(40px) → 0

/* Stagger using --reveal-delay */
<div class="reveal--up" style="--reveal-delay: 100ms">...</div>
<div class="reveal--up" style="--reveal-delay: 200ms">...</div>

Reduced Motion

/* Always include this block — accessibility requirement */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration:   0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration:  0.01ms !important;
    scroll-behavior:      auto !important;
  }
}

Interactive Card Rules

/* All interactive cards follow this pattern */
.card {
  border-radius: 14px;       /* or 16px for value panels */
  border: 1px solid var(--color-border);
  background: var(--color-bg-elevated);
  transition: border-color 0.25s,
              box-shadow    0.25s,
              transform     0.25s ease;
}
.card:hover {
  border-color: rgba(235, 93, 30, 0.3);
  box-shadow: 0 4px 24px var(--color-shadow);
  transform: translateY(-2px);
}

Theme Implementation

/* HTML element carries the theme */
<html data-theme="light">  ← default
<html data-theme="dark">

/* CSS structure */
:root                    { /* light tokens */ }
[data-theme="dark"]      { /* dark overrides */ }

/* Persist to localStorage */
localStorage.setItem('dare-theme', theme);

/* System fallback */
@media (prefers-color-scheme: dark) {
  :root:not([data-theme]) { /* auto dark */ }
}
